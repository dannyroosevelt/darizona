First, set up a free Firebase database. This stores everyone's picks so they sync live. Takes about 3 minutes.

1 Go to console.firebase.google.com and sign in with any Google account
2 Click "Add project" → name it anything → click Continue twice → Create project
3 In the left sidebar click Build → Realtime Database → Create Database
4 Pick any location → select "Start in test mode" → Enable
5 Copy the URL at the top of the page:
https://your-project-default-rtdb.firebaseio.com
